# Prompt 03 — Criação da capa do podcast

## Objetivo

Criar uma capa quadrada, moderna e profissional para o podcast **Segurança em Agentes de IA**.

## Contexto

Este projeto faz parte de uma entrega educacional da DIO para criação de podcast com IA generativa.

Nome oficial do podcast:

Segurança em Agentes de IA

Subtítulo oficial:

LLMs, APIs e Automações Seguras na Prática

Tema central:

Segurança em agentes de IA, LLMs, APIs e automações.

## Tarefa

Crie uma imagem de capa para podcast em formato quadrado.

A capa deve comunicar segurança, tecnologia, agentes de IA, automações e arquitetura em camadas.

## Elementos visuais desejados

- escudo digital;
- cadeado;
- cérebro ou circuito representando IA;
- camadas representando agentes, APIs, automações e dados;
- visual tecnológico limpo;
- composição profissional;
- baixa poluição visual;
- boa legibilidade do título e subtítulo.

## Texto da capa

Título:

Segurança em Agentes de IA

Subtítulo:

LLMs, APIs e Automações Seguras na Prática

## Estilo visual

- capa quadrada;
- estética moderna;
- visual técnico e profissional;
- fundo escuro;
- elementos digitais;
- iluminação sutil;
- contraste alto para leitura;
- aparência adequada para GitHub, LinkedIn e portfólio.

## Regras

- Não usar logos de empresas.
- Não usar marcas registradas.
- Não usar pessoas reais.
- Não usar prints de contas ou ferramentas.
- Não usar dados sensíveis.
- Não usar microfone como elemento principal.
- Não criar visual genérico de tecnologia sem relação com segurança.
- Não mencionar ferramentas específicas na imagem.

## Resultado do Prompt 03

O Prompt 03 gerou a primeira versão da capa com os componentes visuais principais aprovados: escudo, cadeado, cérebro/circuito e camadas relacionadas a agentes, APIs, automações e dados.

A composição foi considerada adequada, mas precisou de correção de paleta para se afastar de uma aparência azul/ciano genérica e aproximar a capa de uma estética dark tech magenta/violeta.
