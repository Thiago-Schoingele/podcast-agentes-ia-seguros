# Prompt 03.1 — Correção da capa com paleta dark tech magenta/violeta

## Objetivo

Corrigir a paleta visual da capa do podcast, mantendo os componentes gráficos aprovados na primeira versão.

## Contexto

A primeira capa do podcast **Segurança em Agentes de IA** trouxe uma composição adequada para o tema, com escudo digital, cadeado, cérebro/circuito e camadas representando agentes, APIs, automações e dados.

Porém, a paleta visual ficou genérica demais, com aparência mais próxima de tecnologia azul/ciano.

## Tarefa

Gere uma nova versão da capa mantendo a estrutura aprovada, mas ajuste a paleta para uma estética dark tech magenta/violeta.

## Manter

- formato quadrado;
- título `Segurança em Agentes de IA`;
- subtítulo `LLMs, APIs e Automações Seguras na Prática`;
- escudo digital;
- cadeado;
- cérebro/circuito;
- camadas com agentes, APIs, automações e dados;
- fundo escuro;
- composição limpa;
- boa legibilidade.

## Corrigir

- reduzir aparência azul/ciano genérica;
- usar mais magenta, violeta e rosa neon;
- manter contraste alto;
- preservar a atmosfera técnica e profissional;
- deixar a capa mais alinhada a uma identidade dark tech.

## Regras

- Não adicionar logos.
- Não adicionar marcas.
- Não adicionar pessoas.
- Não usar microfone como elemento principal.
- Não usar prints de contas.
- Não incluir dados sensíveis.
- Não poluir a composição.
- Não trocar o tema do podcast.

## Resultado do Prompt 03.1

O Prompt 03.1 gerou a capa final aprovada, com paleta dark tech magenta/violeta, título e subtítulo legíveis e os elementos visuais coerentes com segurança em agentes de IA.
